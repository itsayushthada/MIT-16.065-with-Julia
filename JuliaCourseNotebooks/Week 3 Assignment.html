Model Fitting using Cubic Regression¶



First define the least square loss for the model.

$${\large{S_{r}}} = \sum{e^2} = \sum_{i=1}^{n}(y_{i} - (ax_{i}^{2} + bx_{i} + c))^{2} $$
$${\large{S_{r}}} = \sum_{i=1}^{n}((ax_{i}^{2} + bx_{i} + c) - y_{i})^{2} $$

Partially diffrentiate with respect to the parametrs of the model and equate to zero.

$${\large{\frac{\partial S_{r}}{\partial S_{a}}}} = 2\sum_{i=1}^{n}((ax_{i}^{2} + bx_{i} + c - y_{i}).x_{i}^{2}) = 0 $$
$${\large{\frac{\partial S_{r}}{\partial S_{b}}}} = 2\sum_{i=1}^{n}((ax_{i}^{2} + bx_{i} + c - y_{i}).x_{i}) = 0 $$
$${\large{\frac{\partial S_{r}}{\partial S_{c}}}} = 2\sum_{i=1}^{n}(ax_{i}^{2} + bx_{i} + c - y_{i}) = 0$$

Rewrite the equations.

$$ a\sum_{i=1}^{n}x_{i}^{4} + b\sum_{i=1}^{n}x_{i}^{3} + c\sum_{i=1}^{n}x_{i}^{2} = \sum_{i=1}^{n}x_{i}^{2}y_{i} $$
$$ a\sum_{i=1}^{n}x_{i}^{3} + b\sum_{i=1}^{n}x_{i}^{2} + c\sum_{i=1}^{n}x_{i} = \sum_{i=1}^{n}x_{i}y_{i} $$
$$ a\sum_{i=1}^{n}x_{i}^{2} + b\sum_{i=1}^{n}x_{i} + c.n = \sum_{i=1}^{n}y_{i} $$

Now, we can estimate the values of the sum using the given data and represent equations in matrix format.

$$ A^{T}X = B$$

where A in column vector of parameters a, b and c respectively; X is matrix of coefficents and B is the Column Vector of RHS of equations.

$$ A = X^{-1}B $$

Standard error of the estimate is
$$ S_{y/x} = \sqrt{\frac{S_{r}}{n-(m+1)}}, $$

where "n" is total no of data points and "m" is the order of the polynomial.

function update_params(x::Array{Float64,1}, y::Array{Float64,1}, eta::Float64)
    delS_dela = -2 * eta * sum(residue(x, y) .* (x.^2)) 
    delS_delb = -2 * eta * sum(residue(x, y) .* x)
    delS_delc = -2 * eta * sum(residue(x, y))
    
    return [delS_dela delS_delb delS_delc]
end
